云海预测方法：如何根据湿度、低云、风速和逆温判断云海

云海预测不是简单看「晴天」或「湿度高」。可靠做法是同时检查日出窗口 03:00–07:00的湿度、低云/中云、能见度、风速、垂直场（850/700 hPa）以及观景点与谷底/水面的高差关系。

查看实时云海预测

核心判断因子

湿度
谷地近地层越接近饱和，越容易形成雾或低云；模式报 RH 72–85% 时仍可能有观赏级云海。
低云 / 中云
低云决定云海主体；中高云影响日出。NWP 低云≈0 时，需结合能见度推断谷地雾。
风速
微风 ~1–4 m/s 利于维持；强风破坏稳定层。
能见度
300–600 m 常对应谷地雾/层云；与海拔、云底高度一起判断能否「看见脚下云海」。
逆温 / 稳定层
850–925 hPa 正逆温利于填谷；近库辐射雾日上层可偏干，由规则 Type B 补偿。
近库水体
山脚大型水库（如五女山桓龙湖）可触发「水体蒸发型谷地雾」规则，仅规则引擎使用，不进 ML 特征。

规则引擎 + ML v6 如何协作

本站采用混合预测：规则引擎解释气象机理并做能见度/型态补偿；ML 根据历史标注修正本点位偏差。日出窗口内按点位融合（如五女山 ML 约 65%）；近库水体雾、低能见度谷地雾时降低 ML 权重，避免模式低云漏报时 ML 压分。
