右栏预测面板

  • 双概率环：云海（蓝）与日出（橙）；下方为综合场景标签与叙述。
  • Meteogram：可展开温度、湿度/降水、分层云量、风速/阵风/风向四行详图。
  • 场景因子 Tab：ECharts 雷达图，展示规则加权维度。
  • 云海 / 日出 Tab：完整因子列表，含 ML 融合、规则参考与底层观测。
  • 分享：生成链接；OG 横版预览 + 竖版长图（含 Meteogram 与日出窗口表）。
  • AI 出行解读（可选）：基于当前逐小时 JSON 的 LLM 摘要，需服务端配置。
