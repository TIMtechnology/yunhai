云海 · 日出观测概率预测

帮助你在出发前判断某座山、某个观景点未来 5 天（120 小时）内是否值得前往拍摄云海或等待日出。 内置五女山、大黑山、黄山、庐山等精选观景点，也支持通过天地图 POI搜索全国任意地点。

数据时效：每次选点或刷新页面都会重新请求最新预报。你看到的概率对应当前时刻 Open-Meteo 返回的近 5 天逐小时数据，不同小时、不同时间打开页面，数值可能略有变化——这是数值天气预报正常更新，并非 bug。

共建标注：POI → 社区点位 → 提交

欢迎贡献你在现场的云海观测，帮助改进模型。无需注册账号，浏览器会自动生成匿名贡献 ID。

  1. 主页选 POI：左侧搜索并选择天地图 POI（绿色标签），或拖动地图标记到准确位置。
  2. 进入标注：右侧预测面板点击 「标注此点位」，或顶栏 Banner「去标注」。
  3. 自动创建社区点：首次保存标注时，系统将该 POI 注册为社区点位（如 cs_0c28f6f2），500 米内重复坐标会合并。
  4. 选择日期并标注：在 标注页 选择历史日期（仅今天及以前），按 1/2/3 标注云海，并可标注日出质量（可见 / 遮挡 / 不可拍）。
  5. 编辑社区点：在标注页可持久化修改我的社区点名称、坐标、海拔；已落库精选会同步更新。
  6. 提交即生效：社区点位保存后立即 approved；云海标注纳入 ML 训练集，日出质量先入库供后续日出模型训练。
收藏链接：预测页 /?loc=cs_xxx 或 /?lat=…&lng=…&name=…； 标注页 /label.html?loc=cs_xxx。便于下次直达同一社区点。

Admin 审核（维护者）

打开 /label.html?admin=1， 填入 Admin Token 后可：查看待审队列 → 通过/驳回 → 重训 ML → 达标社区点「精选落库」。

  • 仅 approved 标注进入训练集
  • 单社区点 ≥1 天标注后自动「落库」为精选景区
  • 重训通过后需在服务器更新模型 pkl 并重启，03–07 点云海概率才会变化

快速上手（5 步）

  1. 搜索地点：左侧输入景区名（如「云台山」「大黑山」）或点击快捷按钮。绿色标签为天地图 POI，紫色为精选观景点。
  2. 选择观景点：精选景区可切换多个内置观景点（如五女山「点将台」）；POI 结果则直接使用该 POI 坐标。
  3. 微调定位：地图右上角「定位控制」可拖动标记或开启「点击放置」，修正 GPS 偏差后自动重新预测。
  4. 查看概率：右侧显示云海 / 日出双概率环、综合场景（如「较可能日出云海」「晴日日出」）；「场景因子」Tab 为雷达图，「云海」Tab 含 ML、规则与底层观测；可展开 Meteogram 气象详图 核对逐小时场。
  5. 分享与解读：可生成分享链接与竖版长图；开启 AI 出行解读 可对照规则与 ML 输出获得文字摘要（需服务端配置 LLM）。
  6. 可选云图：顶栏开启「云层蒙版」可叠加 Himawari 红外卫星图（随地图缩放自动裁切）。

云海概率

日出窗口 03:00–07:00 由 规则引擎 + ML v6 按点位权重融合（五女山约 65% ML / 35% 规则；近库水体雾、低能见度谷地雾场景会进一步降 ML 权重）；同时展示完整加权因子、底层观测及 Himawari 卫星可选校正。详见 预测模型。

日出概率

在日出前后约 45 分钟窗口，评估低云遮挡、中高云、降水与能见度，结合天文日出时刻给出观测成功率。

常见问题

为什么刷新页面后概率变了？

预报数据来自 Open-Meteo 数值模式（ECMWF 等），模式会定期同化最新观测并更新未来 5 天逐小时场。你刷新页面 = 重新拉取当前最新预报，因此同一时刻（如明早 6 点）的概率可能随模式更新而微调，尤其在 24–48 小时内变化更明显。

每个小时的数值从哪来？

每个小时对应 Open-Meteo 该时刻的湿度、分层云量、风速、降水等，再经本地模糊评分引擎算出云海/日出概率。详见 数据来源说明。

大黑山 / 其他地点定位不准怎么办？

优先在搜索结果中选天地图 POI（绿色标签），或在精选景区下方使用「天地图定位建议」。仍不准时，用地图拖动标记到正确位置，系统会按新坐标重新预测。

卫星云图为什么显示「回溯 N 小时」？

Himawari 红外图只有历史观测。若所选时刻尚无卫星图，系统会自动向前查找最近可用图，并在蒙版说明中标注回溯小时数。
