云海形成条件：湿度、逆温、低云和风速如何影响云海

观赏级云海通常出现在：山谷湿度充足、观景点高于云雾层、风速不强，且日出前后为最佳判断窗口。本站以 03:00–07:00 作为云海与日出联合预测的核心时段。

关键条件

湿度充足：近地层 RH 高时易形成雾或低云；模式报 RH 72–85% 仍可能有云海（尤其配合低能见度）。
逆温或稳定层：有利于云雾在山谷水平铺展；但近库辐射雾日上层可偏干，不一定需要强逆温。
低云 / 能见度：模式低云偏低时，能见度 300–600 m 常是谷地雾/层云的直接证据。
近库大型水体：山脚水库蒸发 + 谷地冷却可形成填谷型雾（五女山桓龙湖等）；本站通过规则 Type B 识别，不依赖 ML。
弱风：微风利于维持；强风破坏稳定结构。
日出遮挡：中高云过多会挡太阳；需同时看日出概率与云海概率。

如何用本站验证

在预测页展开 Meteogram，查看日出窗口的湿度、云量与风；阅读因子面板中的「云海型态」与 ML/规则对照；使用分享图与 日出云海指南 综合决策。
