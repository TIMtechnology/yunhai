预测模型说明

基于 Open-Meteo 逐小时场（含 850/925/700/500 hPa 气压层）与可选 Himawari 卫星区域云量，采用加权模糊评分。每个预报小时独立计算，故时间轴各点概率可以不同。

Phase A · 垂直场因子（文献化）

主要依据： 《基于多源立体探测和模糊逻辑算法的北京高山云海预报研究》 ——「下湿上干」、850 hPa 以下 RH>80%、925–850 hPa 逆温、模糊逻辑加权。
