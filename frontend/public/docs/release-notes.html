v7.1 · 贴地雾校准 & 漏报修复

在 6 月初 ML v7 上线基础上，针对「高湿但无观赏云海」「NWP 低云=0 却有云海」等边界 case 优化规则与融合逻辑，并重训五女山模型。5–6 月标注回测方向一致率由 83% 提升至 88%。

你会感受到什么变化？

  • 更少「假高概率」：清晨高湿、能见度差但并无层云时（贴地雾/霾），概率不再被抬到 60–70%。
  • 真云海日更稳：近库低能见度、饱和谷地云等 Type B 场景召回改善。
  • 日出后查看更可信：当天 7 点后再打开页面，日出窗内各时刻概率已冻结，不会因能见度修正突然跳变。
  • Admin 标注页：历史预测访问面板支持双轨曲线、分段差异与诊断（若已开 admin）。
