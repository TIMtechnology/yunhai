Meteogram 怎么看：用气象详图判断云海与日出概率

Meteogram 是把逐小时天气压缩成一张气象详图。本站的 Meteogram 用于解释云海概率和日出概率：它展示温度变化、湿度和降水、分层云量、风速阵风以及风向。

四个面板分别看什么

温度
夜间降温和日出前冷却有利于近地层水汽凝结。
湿度 / 降水
湿度高说明水汽条件较好，前期降水可能提供水汽补给。
分层云量
低云更接近云海主体，中高云更多影响日出遮挡。
风速 / 阵风 / 风向
弱风利于云雾维持，阵风大时云层容易被破坏。

为什么要配合 ML 和规则引擎

Meteogram 展示的是气象事实，规则引擎把事实转成可解释评分，ML 模型再根据历史标注修正点位偏差。三者结合，比单看天气图标更适合判断日出云海。
